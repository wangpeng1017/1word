## 常见问题排查

### 登录失败
1. 检查网络配置是否正确
2. 查看控制台 Network 标签的请求
3. 确认 API 地址配置：`app.js` 中的 `apiUrl`
4. 测试账号是否正确：`2025001` / `123456`

### 图标不显示
1. 检查 `images` 目录下是否有图标文件
2. 重新编译小程序
3. 清除缓存后重新编译

### 数据加载失败
1. 检查后端服务是否正常运行
2. 查看控制台错误信息
3. 确认 token 是否有效
4. 尝试重新登录

### 音频播放失败
1. 检查后端是否返回了 `audioUrl`
2. 确认音频地址可访问
3. 查看控制台音频加载错误
