## 服务器域名配置

在微信开发者工具中需要配置以下服务器域名：

### 开发阶段（本地测试）
1. 点击右上角 "详情" 按钮
2. 找到 "本地设置"
3. 勾选 "不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"

### 正式发布前
需要在微信公众平台配置服务器域名：

1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入 "开发" -> "开发管理" -> "开发设置"
3. 找到 "服务器域名" 配置
4. 添加以下域名：

**request 合法域名：**
```
https://11word.vercel.app
```

**注意事项：**
- 域名必须使用 HTTPS 协议
- 每月只能修改 5 次
- 域名需要备案（如果是国内服务器）
- Vercel 域名自带 SSL 证书，无需额外配置

## API 地址切换

如需在本地环境和线上环境之间切换，修改 `app.js` 中的 `apiUrl`：

```javascript
// 本地环境
apiUrl: 'http://localhost:3000/api'

// 线上环境（当前配置）
apiUrl: 'https://11word.vercel.app/api'
```

## 常见问题

### 1. 登录失败
- 检查是否勾选了 "不校验合法域名"
- 确认 API 地址是否正确
- 查看控制台是否有网络错误

### 2. 网络请求失败
- 检查服务器域名配置
- 确认后端服务是否正常运行
- 查看微信开发者工具控制台的网络请求详情

### 3. Token 过期
- 重新登录即可
- Token 有效期由后端配置决定
