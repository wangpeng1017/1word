- `study-green-theme.wxss` - 学习页样式绿色主题备份

## 如何恢复绿色主题

如需恢复绿色主题，执行以下命令：

```bash
# 在 wechat-miniapp 目录下执行
copy backups\app-green-theme.wxss app.wxss
copy backups\index-green-theme.wxss pages\index\index.wxss
copy backups\login-green-theme.wxss pages\login\login.wxss
copy backups\study-green-theme.wxss pages\study\study.wxss
copy backups\wrong-green-theme.wxss pages\wrong\wrong.wxss
copy backups\profile-green-theme.wxss pages\profile\profile.wxss
```
