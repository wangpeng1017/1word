## 📝 API测试示例

### 学生登录
```bash
curl -X POST http://localhost:3000/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "studentNo": "2025001",
    "password": "123456"
  }'
```

### 获取今日任务
```bash
curl -X GET "http://localhost:3000/api/daily-tasks?studentId=STUDENT_ID" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

### 获取词汇题目
```bash
curl -X GET "http://localhost:3000/api/vocabularies/VOCAB_ID/questions" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## 🐛 故障排查

### 问题：无法登录
- 检查学号是否正确
- 检查密码是否为 `123456`
- 确认数据已初始化

### 问题：没有今日任务
- 运行验证脚本检查数据
- 确认任务日期是今天
- 重新运行初始化脚本

### 问题：题目显示异常
- 检查词汇是否有4种题型
- 每题是否有4个选项
- 运行验证脚本
